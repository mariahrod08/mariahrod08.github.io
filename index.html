Version Control System

Version Control System (VCS) is just software that helps you control the different versions of something (typically source code). A detailed history of the project will be maintained as well as the ability to work on different versions of it.

In this tutorial, we’ll be using Git which is a distributed version control system.

Why should you learn this system?

  • Version control repos lie at the heart of most modern-commercials and open-source software projects, as they allow multiple developers to share files in a safe, controlled way, while automatically creating a history of the project’s evolution.
  • On grasping the basics of VCS, students can take advantage of the tool later on.
  • Knowing that the code will be published on a public website would help the students to write better code.
